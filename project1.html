Project Overview

This project focuses on accurately classifying brain MRI images into four categories: Glioma, Pituitary, Meningioma, and No Tumor. It leverages deep learning, specifically a custom CNN model (along with VGG16 and VGG19), to achieve high-performance classification.

To enhance transparency and trust in AI predictions, Explainable AI (XAI) techniques using LIME are integrated, highlighting key regions in the MRI images that influenced the model's decision.

A Streamlit-based user interface allows users to easily upload MRI scans and obtain both the classification result and a visual explanation, making the system intuitive and practical for real-world applications.

Key Highlights:

  • Custom CNN for accurate tumor classification
  • VGG16 and VGG19 for comparative analysis
  • LIME-based heatmaps for interpretability
  • Interactive Streamlit UI for user-friendly predictions
  • Published as a research paper in Springer
