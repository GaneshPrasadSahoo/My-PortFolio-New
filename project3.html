Project Overview

Health Metrics Prediction is a machine learning project designed to estimate key health indicators such as Blood Pressure (Systolic/Diastolic), Stress Level, and Heart Rate based on user inputs like Gender, Age, Sleep Duration, Physical Activity Level, Daily Steps, and BMI Category.

Built with Streamlit, this interactive web application allows users to input their details and instantly receive predictions for their vital health metrics.
