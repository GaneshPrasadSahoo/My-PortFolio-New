Project Overview

BIO GPT is an AI-powered web application designed for biology students, providing accurate question-answering and content expansion. The system is trained on NCERT Biology textbooks to ensure precise and relevant explanations. It leverages Google Gemini, HuggingFace models, and LangChain to deliver both retrieval-based answers and dynamically expanded content.

Key Features:

  • Ask questions directly related to NCERT Biology.
  • Get short, accurate answers using a retrieval-based approach.
  • Expand answers dynamically using Google Gemini AI.
  • Choose answer length: 100 words, 200 words, or custom.
  • Interactive, centered, chat-like interface with a dark theme.
  • Built with Streamlit for easy deployment and UI customization.
